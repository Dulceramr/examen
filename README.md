## Procesamiento y Clasificación de Datos en Angular

Este proyecto muestra un ejemplo práctico de procesamiento, transformación y clasificación de datos dentro de un componente **standalone** en Angular. Su propósito es evaluar habilidades fundamentales en manipulación de datos, buenas prácticas con componentes, uso de `OnInit`/`AfterViewInit` y separación lógica mediante filtros.
